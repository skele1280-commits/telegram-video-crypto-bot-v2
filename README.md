# Telegram Video & Crypto Bot

A professional Telegram bot for downloading videos from supported links and providing real-time cryptocurrency market data.

## Features
- **Video Download**: Use `/download <url>` to fetch videos (e.g., YouTube).
- **Crypto Prices**: Use `/crypto` for top 10 market caps and the top gainer.
- **Scheduled Updates**: Use `/updates 15m` or `/updates 1h` for automatic crypto updates.

## Setup
1. Get a bot token from [@BotFather](https://t.me/botfather) on Telegram.
2. Set `BOT_TOKEN` in your environment (e.g., Replit secrets).
3. Install dependencies: `pip install -r requirements.txt`
4. Run: `python main.py`
